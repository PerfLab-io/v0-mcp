# v0.dev MCP Server

An MCP (Model Context Protocol) server that exposes v0.dev APIs for AI-powered development tools.

## Setup

```bash
npm install
npm run start
```

## Adding to Claude Desktop

### 1. Get v0.dev API Key
Get your API key from https://v0.dev/chat/settings/keys

### 2. Deploy to Vercel
```bash
npm run deploy
```
